# 🍽️ Restaurant Reservation System

A Windows Desktop application developed using **C# and .NET** to manage restaurant reservations efficiently and organize customer bookings بطريقة سهلة وسريعة.

---

## 🧾 Description

The Restaurant Reservation System is designed to help restaurant staff manage table bookings and customer data بشكل منظم.

The application allows users to create, update, and delete reservations, مع إمكانية متابعة حالة الحجز وتجنب التعارض بين المواعيد.
It simulates a real-world restaurant management scenario and focuses on simplicity, usability, and performance.

---

## 🚀 Features

* ➕ Add new reservations
* ✏️ Edit existing reservations
* ❌ Delete reservations
* 👥 Manage customer data
* 🪑 Track table availability
* 📋 View all bookings in an organized way

---

## ⚙️ How to Run the Project

1. Clone the repository:

```id="run1"
git clone https://github.com/RamezSameh/Restaurant-Reservations.git
```

2. Open the project in **Visual Studio**

3. Make sure **SQL Server** is running

4. Update the database connection string in the project

5. Run the application
